## Description
Switch primary screen between any number of monitors using GUI

## Requirements
- bash
- xrandr
- zenity

## Setup
`cp swprimon-gui.sh ~/.local/bin/swprimon-gui`

Note: make sure `~/.local/bin/` exists and in PATH

## Usage
Set keboard shorcut from Settings -> Keyboard
